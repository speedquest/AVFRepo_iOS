Week 1 Discussion Post

What types of native features (on Android and iOS devices) would you be able to implement with the use of HTML/CSS/JS only? (just a hint - iOS can be the tricky one)

Based on my research, since we are using the PhoneGap software, the only native features we would not be able to access are the compass on older iPhone models.

For your CRUD app revision, which of these do you think you will implement?

I would like to implement the Geolocation, Notification, and camera for my CRUD app.

Briefly describe some past 'lessons learned' regarding workflow as it relates to research, design, development, etc.

I have definitely learned that utilizing some sort of revision control is 100% necessary. I am still not 100% sold on GitHub, as it still gives me problems.

Please describe any design intentions you may have for the AVF Demo app and the revised/new version of your CRUD app.

My app is supposed to be used to track your automobiles service intervals, fuel mileage, etc. In a perfect world, it would track multiple vehicles and allow users to switch between those vehicles at will. It would use the Geolocation to determine miles travelled, and would use Notification feature to alert users about important service milestones, etc. Addtionally, I would love to be able to use the camera to scan the barcode VIN numbers that so many modern vehicles are equipped with nowadays.
